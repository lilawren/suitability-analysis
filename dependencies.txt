This file lists external dependencies of this project that need to be installed on
the server running this system:

ArcGIS Pro - running Python 3.4 with arcpy or ArcGIS 10.3 and up running Python 2.7 with arcpy (3.4 compatibility first, followed by 2.7)
GDAL - see https://pypi.python.org/pypi/GDAL/ section about installing on Windows
